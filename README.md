# ESX Food Delivery Job

Adds Food Delivery Job to  Five M Servers running ESX Framework

## Requirements
- Ox Target


## Download & Installation

### Using Git
```
cd resources
git clone https://github.com/riostokes/ESX-Food-Delivery-Job
```

### Manually
- Download https://github.com/riostokes/ESX-Food-Delivery-Job
- Put it in the `[esx]` directory

## Installation
- Import `rs_food_delivery_job.sql` in your database
- Add this in your `server.cfg`:

```
start rs_fooddelivery
```
